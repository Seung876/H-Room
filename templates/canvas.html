빈 강의실 실시간 조회 서비스

Problem Statement Canvas

1) Problem Statement Canvas

고객

한국외국어대학교 재학생

컨텍스트

수업 외 시간에 자습·스터디·회의를 할 공간이 필요하지만 ‘빈 강의실’을 한눈에 확인할 수 있는 시스템이 없음

근본 문제

  • 대여로만 사용가능한 강의실도 있음/ 정보 미통합 → 실제 공실 여부 확인 어려움
  • 학생이 직접 건물마다 탐색해야 해서 시간이 낭비된다.

감정적 영향

  • 불편함, 짜증
  • 시간 낭비
  • 비효율감

정량적 영향

  • 평균 10분 이상 탐색 시간 손실
  • 시험기간(피크) 최대 1시간 손실
  • 공간 활용률 저하(공실 인지 실패)

대안

  • 학사포털 수동 검색
  • 학지팀문의
  • 학생 커뮤니티 질문

대안의 단점

  • 실시간성 부족
  • 정보 분산
  • 학기마다 수동 확인 필요

2) 솔루션 (3줄)

Line 1: 학사포털 시간표 데이터를 자동 수집하여 요일·시간대 기준으로 ‘현재 비어 있는 강의실’을 한눈에 제공
Line 2: 수업·대여 강의실 여부를 통합해 실제 사용 불가능한 강의실을 자동 필터링
Line 3: 사용자가 건물·요일·시간을 선택하면 즉시 이용 가능한 강의실을 한눈에 확인가능

3) 기능 명세

  • 강의시간표 자동 크롤링
  • 요일/시간/건물 기준 공실 조회
  • 대여여부 CSV 통합 반영
  • 상태 구분: 공실 / 대여가 필요한 강의실 / 수업중
  • 즐겨찾기 등록 및 최근 검색 유지
  • 데이터 기준 시각 표시 (갱신 시간 명시)

4) 간단한 UX 흐름

  1. 메인 페이지 – “지금 비어있는 강의실 보기” 버튼
  2. 필터 영역 – 요일 / 시간 / 캠퍼스 / 건물 선택
  3. 결과 화면
    • 리스트뷰: 강의실명, 상태(공실/수업중/대여가 필요한 강의실), 다음 수업 시간
    • 지도뷰(선택): 공실 위치 표시
  4. 상세 페이지 – 사진, 좌석 수, 다음 수업 시간, 대여 여부
  5. 즐겨찾기 탭 – 자주 사용하는 강의실 저장 및 빠른 접근

※ 자연어 검색(“오늘 2시에 공학관 비는 방”)은 메인 하단 입력창에서 지원
