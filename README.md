## H:ROOM

한국외국어대학교 재학생을 위한 공실 조회 웹 서비스

---

### 프로젝트 소개

**H:ROOM**은 학사포털의 시간표, 대여 여부 데이터를 통합하여  
사용자가 **요일·시간·건물 기준으로 현재 비어 있는 강의실을 한눈에 확인**할 수 있도록 돕는 웹 서비스입니다.  

기존에는 수업 여부만 확인 가능해 학생들이 직접 강의실을 찾아야 했지만
H:ROOM은 이러한 불편을 해소하고 **공간 활용 효율을 높이는 것**을 목표로 합니다.

---

### 주요 기능

- 요일 / 시간 / 건물 기준 공실 조회
- 건물별 공실률 시각화 그래프 제공
- 대여정보 통합 반영 (수업·대여·공실 상태 구분)
- 즐겨찾기
